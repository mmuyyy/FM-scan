# Industrial FPV Scanner

## 项目概述
工业级 FPV 和 FM 信号检测脚本，用于自动扫描、检测、解调和解码 5.8G FPV 视频信号和 FM 广播信号。

## 功能特点

### 核心功能
1. **多频段扫描**：支持同时扫描 FM 广播频段 (87.5-108 MHz) 和 5.8G FPV 频段 (5725-5925 MHz)
2. **信号检测**：基于带宽、SNR 和星座图分析检测 FM 信号
3. **FPV 信号识别**：通过视频同步信号 (15625 Hz) 识别 FPV 图传信号
4. **FM 解调**：使用相位差分法进行 FM 解调
5. **实时可视化**：实时显示频谱、时域信号和扫描进度
6. **数据存储**：自动保存检测到的信号信息、原始样本和解调数据
7. **系统监控**：实时监控系统状态和错误处理

### 工业级特性
1. **高可靠性**：多路径 API 支持，确保兼容不同版本的 UHD
2. **错误处理**：完善的错误处理和故障恢复机制
3. **性能优化**：多线程并行处理，提高扫描效率
4. **资源管理**：高效的内存管理和资源使用
5. **24/7 运行**：支持持续扫描模式，适合长时间运行
6. **详细日志**：完整的日志记录，便于故障排查

## 技术栈
- Python 3.8-3.11
- UHD (Universal Hardware Driver)
- NumPy
- SciPy
- Matplotlib
- PyAudio (可选，用于音频输出)

## 硬件要求
- USRP B210 或 B200mini
- 合适的天线（FM 频段和 5.8G 频段）
- 计算机运行 Windows 或 Linux

## 安装指南

### 1. 安装依赖

#### Windows
```bash
# 安装 Python 3.8+ (推荐 3.10)
# 安装 UHD 驱动
# 安装 Python 包
pip install numpy scipy matplotlib pyaudio
```

#### Linux
```bash
# 安装 UHD 驱动
sudo apt-get install libuhd-dev uhd-host

# 安装 Python 包
pip install numpy scipy matplotlib pyaudio
```

### 2. 配置 USRP 设备
- 连接 USRP 设备到计算机
- 确保 UHD 驱动能够正确识别设备
- 运行 `uhd_usrp_probe` 验证设备连接

## 使用方法

### 1. 基本使用
```bash
# 运行工业级 FPV 扫描器
python industrial_fpv_scanner.py
```

### 2. 配置参数
编辑 `config.json` 文件可以修改以下参数：

```json
{
  "sample_rate": 2000000,       # 采样率 (Hz)
  "gain": 30,                   # 增益 (dB)
  "antenna": "RX2",            # 天线选择
  "bands": [                    # 扫描频段
    {
      "name": "FM Broadcast",  # 频段名称
      "start": 87500000,        # 起始频率 (Hz)
      "stop": 108000000,        # 结束频率 (Hz)
      "step": 100000            # 步进频率 (Hz)
    },
    {
      "name": "5.8G FPV",
      "start": 5725000000,
      "stop": 5925000000,
      "step": 1000000
    }
  ],
  "scan_params": {
    "num_samples": 8192,        # 每次接收的样本数
    "timeout": 1.0,             # 接收超时时间 (秒)
    "continuous": true          # 是否持续扫描
  },
  "visualization": {
    "fft_size": 1024,           # FFT 大小
    "window_size": 50,          # 显示窗口大小
    "update_interval": 50       # 更新间隔 (毫秒)
  }
}
```

## 数据流

1. **设备初始化**：配置 USRP 设备参数
2. **数据采集**：从 USRP 接收 IQ 样本
3. **信号处理**：计算带宽、SNR 和星座图分析
4. **信号检测**：识别 FM 和 FPV 信号
5. **解调**：对检测到的信号进行 FM 解调
6. **数据存储**：保存信号信息和数据
7. **可视化**：实时显示信号特征
8. **系统监控**：监控系统状态和错误

## 故障排查

### 常见问题

1. **设备初始化失败**
   - 检查 USRP 设备连接
   - 验证 UHD 驱动版本
   - 检查权限设置

2. **数据接收失败**
   - 检查天线连接
   - 调整增益设置
   - 检查采样率设置

3. **信号检测不准确**
   - 调整扫描步进频率
   - 检查 SNR 阈值设置
   - 验证信号源强度

4. **可视化卡顿**
   - 降低 FFT 大小
   - 增加更新间隔
   - 关闭不必要的应用程序
